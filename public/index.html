💡 À propos des backups
📸 Un backup est un snapshot JSON complet de la structure Discord : rôles, catégories, salons, topics.
💾 Les fichiers sont sauvegardés localement sur le serveur Railway sous forme de backup_XXXX.json
🔄 Pour restaurer un backup, il faudra copier son contenu dans discord-template.json et lancer une sync F→D.
💡 Le README mentionne une backup auto toutes les 6h — vérifie que tu as bien implémenté le setInterval dans server.js !
